### Komanda **mkdir**

pravljenje foldera

```
mkdir
```

```
mkdir folder/subfolder/subsubfolder
```

pravljenje parent directorija, koristi se u slucaju da parent folder ne postoji te zelimo u isto vrijeme napraviti parent folder te u njemu subfoldere

```
mkdir -p
```
